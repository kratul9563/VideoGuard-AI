# VideoGuard AI

VideoGuard AI is an enterprise-grade Generative AI application designed for automated video compliance auditing using Retrieval-Augmented Generation (RAG) and Agentic AI workflows.

## Tech Stack

* Azure OpenAI (GPT-5)
* Azure AI Search
* Azure AI Video Indexer
* LangGraph
* LangChain
* FastAPI
* LangSmith
* Azure Application Insights

## Features

* Video ingestion from YouTube
* Transcript extraction and OCR processing
* RAG-based compliance retrieval
* AI-powered compliance reasoning
* Automated violation detection
* Structured JSON compliance reports
